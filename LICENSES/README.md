- Hardware design files (CAD/SCAD/STL): CERN Open Hardware License v2 variants
  - Strongly reciprocal: CERN-OHL-S-2.0
  - Weakly reciprocal: CERN-OHL-W-2.0
  - Permissive: CERN-OHL-P-2.0
- Documentation (text, images): Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)

References:
- CERN-OHL family: https://ohwr.org/project/cernohl/wikis/Documents/CERN-OHL-version-2
- CC BY-SA 4.0: https://creativecommons.org/licenses/by-sa/4.0/

Notes:
- Unless otherwise stated in a subfolder, STL exports and source CAD in `hardware/` are covered by CERN-OHL-S-2.0 by default for builds.
- You may additionally use W or P variants for derivative distribution as needed; see license texts for conditions.
